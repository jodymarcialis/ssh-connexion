# ssh-connexion
![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

A simple script that creates an SSH connexion to a server (using pxssh), and executes bash commands with subprocess. If you are connected to your local network, you just have to change the login values to your server name, your personnal ID and password.

When you are connected, the ls and free -h commands are executed to make sure the connection works, but feel free to modify the code and add scripts to use your own module
